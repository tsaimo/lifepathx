# lifepathx

Vue 3 + Vite 最小项目脚手架，仅包含一个 main 页面。

## 目录结构

```text
lifepathx/
├── .gitignore          # 忽略依赖与构建产物
├── index.html          # Vite HTML 入口
├── package-lock.json   # 锁定依赖版本
├── package.json        # 脚本与依赖
├── src/
│   ├── main.js         # Vue 应用入口
│   ├── App.vue         # 根组件
│   └── pages/
│       └── MainPage.vue # main 页面
└── vite.config.js      # Vite 配置
```

## 命令

```bash
npm install
npm run dev
npm run build
```
